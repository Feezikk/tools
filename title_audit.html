Title Rules
Character limit: Title Character Count cannot exceed 100 characters. Validated automatically in CS Source.

N/A Override: By default, all downstream tabs (2-7) strictly evaluate every row entered in CS Source. If a specific field legitimately does not require a match, manually type N/A into that downstream cell to bypass the check.

Multi-State Mode
Adds a State column to CS Source and all downstream sections for courses that run in multiple states (e.g. TX, FL). The state entered in CS Source is visible throughout all validation tabs.
Multi-Part Exam Tracking
Enables a Multi-Part checkbox on each row. Parts share a parent title but track individual time and point values.
