Table Editor Quick Guide

        

            

                
Workflow

                
Click and drag inside the table grid to highlight multiple cells. Actions like Merging, Splitting, Clearing Content, and Formatting apply to the entire highlighted selection.

            

            

                
Buttons

                
    
                    
  • Structure: Toggle special table sections (Headers, Captions, Footers).
    • 
                    
  • Add/Remove: Add or delete rows/columns relative to your cursor position.
    • 
                    
  • Merge: Combine highlighted cells into one large cell.
    • 
                    
  • Split: Revert a merged cell back into individual cells.
    • 
                

            

            

                
Keyboard Commands

                
    
                    
  • Undo Action inside Table Editor Ctrl+Z
    • 
                    
  • Insert Row Above / Below Alt + ↑ / ↓
    • 
                    
  • Insert Col Left / Right Alt + ← / →
    • 
                    
  • Delete Row Alt + Backspace
    • 
                    
  • Delete Column Alt + Shift + Backspace
    • 
                    
  • Clear Content in Highlighted Cells Delete
